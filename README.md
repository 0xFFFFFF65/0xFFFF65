## **其他**
- ### **开源项目使用“类型-名称-用处”命名规则**  
- ### **开源项目里的变量，函数......使用*匈牙利命名法*,如下：（除了一些特殊的类型会用下划线命名法）**
  
  > 变量：
  > 
  > ```
  > std::string s0xFFFFFF65;
  > ```
  > 
  > 函数：
  > 
  > ```
  > void v0xFFFFFF65();
  > ```
  > 
  > 类：
  > 
  > ```
  > class C0xFFFFFF65
  > {
  > public:
  > 	void v0xFFFFFF65();
  > public:
  > 	std::string s0xFFFFFF65;
  > };
  > ```
